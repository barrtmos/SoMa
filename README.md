# SoMa — Сервис знакомств

![SoMa Screenshot](./assets/screenshot.png)

Современный и стильный сервис знакомств, построенный на передовых веб-технологиях.

## Стек технологий

*   **Frontend**: React (с использованием Vite)
*   **Стиль**: Vanilla CSS (с поддержкой переменных и современных эффектов)
*   **Backend**: Supabase (Auth, Database)
*   **Роутинг**: React Router

## Инструкция по запуску

1.  **Установка зависимостей**:
    ```bash
    npm install
    ```

2.  **Настройка окружения**:
    Создайте файл `.env` в корне проекта и добавьте свои данные Supabase:
    ```env
    VITE_SUPABASE_URL=ваш_url
    VITE_SUPABASE_ANON_KEY=ваш_ключ
    ```

3.  **Запуск в режиме разработки**:
    ```bash
    npm run dev
    ```

4.  **Сборка проекта**:
    ```bash
    npm run build
    ```

## Функционал
*   Главная страница с современным дизайном.
*   Система регистрации и авторизации через Supabase.
*   Личный кабинет пользователя.
